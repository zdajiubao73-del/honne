**起業の科学より：タイミングは成功要因ランキング1位（42%）。「なぜ今なのか」に明確に答えること。**

```

S（社会）: 日本の孤独問題の深刻化→孤独・孤立対策推進法（2023年施行）
P（政治）: 厚労省のメンタルヘルス対策強化施策 → 法人需要の顕在化
E（経済）: コロナ後の在宅勤務定着 → 職場の「雑談」機会の喪失が常態化

→ 「法的追い風 × AI技術成熟 × 孤独の社会問題化」が2024-2026年に同時に起きている
   これが「Why Now」の本質。PLAN.mdに明記すべき。

```

   （3ヶ月後に模倣不可能になるモート）

**要確認：あなた自身がこの課題を「自分ごと」として体験しているか？**

## 2. CPF（課題の質を上げる）の評価 ← 最重要

### 現状：インタビュー0人で設計が進みすぎている

PLAN.mdのペルソナ「田中さやか（27歳）」は**あくまで仮説**。
起業の科学は「最低20人以上にインタビューすること」を条件としている。

### エバンジェリストカスタマーを今すぐ探す

```
エバンジェリストカスタマーの5条件（本書より）：
✓ 課題を認知している
✓ 積極的にソリューションを求めている
✓ 現状は代替手段で解決しようとしている（鍵垢・日記・我慢）
✓ 月980円なら払える
✓ 初期の荒削りなプロダクトでも使いたい

→ 探す場所の具体案：
   - Twitterで「誰にも言えない」「聞いてほしい」「吐き出したい」で検索
   - Redditの日本語版・5chの「メンタルヘルス」板
   - 20-30代の知人・友人で一人暮らしをしている人
   - Pococha / ライブ配信で「聞いてほしいことがある」とコメントしている人
```

### 今すぐ実施すべきインタビュー設問（プロブレムインタビュー）

```
NG: 「このアプリ使いますか？」（解決策へのYes/Noは無意味）
OK: 以下の質問リスト

1. 最近、誰かに話したかったのに話せなかった経験はありますか？
2. そのときどうしましたか？（日記・SNS・ひとりで抱えた？）
3. 今使っている方法の一番の不満は何ですか？
4. 理想的には、どんな方法で解決したかったですか？
5. 「AIに話しかける」ことに抵抗はありますか？あるとしたらなぜ？
6. 月980円で悩みを聞いてくれるサービスがあったら使いますか？
   → 「Yes」と言っても信じない。「実際に今日から使える状態になったら登録しますか？」まで聞く

インタビューの本音抽出ルール（本書より）：
「言動」→「本音」→「深層の本音」の3層で分析する
例：
  言動: 「AIでも話せると思う」
  本音: 「人に知られたくないだけで、実は相手は何でもいい」
  深層: 「受け入れてもらえるかが怖いだけ」
```

### CPF完了条件（本書の基準）

```
□ 20人以上にインタビュー完了
□ 課題の存在を確認（「頻度・強度・代替手段の不満」が言語化できた）
□ ペルソナの修正・精緻化
□ 想定と違った発見を3つ以上言語化する
```

## 4. PMF（製品市場適合）の評価

### AHAモーメント（マジックナンバー）が未設計 ← 要追加

**本書より：Twitter「最初の3日で10人フォロー」= 定着の指標**

```
honneのAHAモーメント候補：

仮説①「初回セッション完了後に『また来る』と感じた人が定着する」
  → 測定方法: オンボーディング→初回チャット完了率 & 48時間以内の2回目起動率

仮説②「3日連続でチャットした人は30日後も使っている」
  → ストリーク機能はこれを促進するための正しい設計

仮説③「整理フェーズまで到達した会話（10往復以上）は継続につながる」
  → AIの会話品質がAHAモーメントを決定する

→ リリース後すぐに定着ユーザーと離脱ユーザーの初期行動を比較分析する
```

### ショーン・エリス・テストの設計

```
PMF達成の判断基準：
「このアプリがなくなったらどう思いますか？」
→ 40%以上が「非常に残念」= PMF達成

TestFlight の10人段階から計測を開始する。
目標: 40%到達まではスケールしない（広告・PR禁止）
```

### AARRR指標の設計（追加）

```
Acquisition（獲得）: App Store impressions → DL CVR
  目標: 5%（REFINED.mdの最適化後推定値）
  計測: App Store Connect のAnalytics

Activation（活性化）: DL → オンボーディング完了 → 初回チャット完了
  目標: 80%完了（REFINED.mdの推定値）
  → AHAモーメントはここで計測

Retention（定着）: 7日・30日・90日リテンション率
  目標: 7日40%以上でないとPMF前と判断
  → これが最重要指標。Paywallより優先

Referral（紹介）: 口コミ・SNSシェア
  → 「今日話してよかった」感想を自然にシェアしたくなるUX設計
  → セッション終了後「今日の一言」カード（Instagramシェア用）

Revenue（収益）: 無料→Pro転換率
  目標: 22%（REFINED.md）
  → リテンションが安定してから課金圧力をかける
```

## 5. 収益計画の修正

### 現在の計画の問題点

PLAN.mdの収益シミュレーション：
```
ダウンロード50,000 × 転換率5% = 2,500人 → MRR 245万円
```

**問題①：CPF未完了の状態での数値は意味がない（本書の警告通り）**

**問題②：転換率5%は楽観的すぎる**

```
実際のベンチマーク：
- App Storeメンタルヘルスカテゴリの無料→有料転換率: 1-3%（業界平均）
- REFINED.mdの最適化後推定: 22%
→ 22%はリテンション×Paywall最適化後の理想値。初期は3-5%から始まる

修正シミュレーション（保守的）：
  DL 50,000 × オンボーディング80% × 7日リテンション55% × 転換率5%
  = 1,100人 → MRR 約108万円（初年度現実的な上限）

修正シミュレーション（楽観的）：
  DL 50,000 × 転換率22% = 11,000人 → MRR 約1,078万円
  ※ これはPMF達成後かつ大規模集客ができた場合
```

**問題③：法人プランがPMF前に登場している**

本書：「広い市場を最初から狙う（ニッチ市場を制圧してから拡大すべき）」
→ 法人プランはv2.0以降・個人でのPMF達成後に着手すること。今は削除。

### LTV/CPA計算（追加）

```
LTV計算（SaaS型）:
  ARPU（月額平均） = ¥980（Pro）
  Churn Rate（目標） = 5%/月（= 平均継続期間20ヶ月）
  LTV = ¥980 / 0.05 = ¥19,600

CPA目標:
  LTV > CPA × 3 が理想
  → CPA上限 = ¥19,600 / 3 = 約¥6,500

App Store 広告（ASA）のCPI: 約¥200-500
オーガニック（SNS口コミ）のCPI: 約¥0-50
→ 初期はオーガニックのみでLTV/CPA比を安全圏に保つ
→ 有料広告はLTV/CPA > 3が確認できてからのみ実施
```

## 7. 販売プランの推敲

### App Storeのポジショニング修正

REFINED.mdの内容は優秀だが、1点追加：

**「これは医療・セラピーではない」を先に言う**

```
App Storeの説明文（推奨順序）：

1行目: 「夜、誰かに話を聞いてほしくなることはありますか。」← そのまま維持

追加すべき一文:
「カウンセリングでも相談でもない。ただ、話を聞いてもらう場所。」

理由: 「メンタルヘルスアプリ」と聞いた瞬間に「自分は病気じゃない」と
     ダウンロードを躊躇するターゲットが多い（確証バイアス）
     → honne はカジュアルで日常的なものだと先に伝える
```

### 価格設定の再検討

```
現状: ¥980/月 or ¥7,800/年

起業の科学的視点での問題:
PMF前に価格を固定するのはリスク

推奨:
  Phase 2-3（TestFlight）: 無料（代わりに週次フィードバックをもらう）
  Phase 4（App Store）初期: ¥480/月（低摩擦で最初のPro会員を50人確保）
  Phase 4 3ヶ月後: ¥980/月（既存ユーザーは維持価格で継続）

根拠：
  「MSP（Minimum Sellable Product）は売れることのほうが簡単で明確な証明方法」
  まず1人目が課金することを証明することが最初のゴール
```

### コンテンツマーケティング（CPA削減）

```
MVC（Minimum Viable Campaign）として先に試すもの：

① ビルドインパブリック（コスト¥0）
  「開発者がアプリを作る過程を公開する」
  → X（Twitter）で週1回投稿「今週honneにこんな機能を入れた理由」
  → 共感を呼び、インタビュー協力者・初期ユーザーを獲得

② ターゲット向けコンテンツ（コスト¥0-5万）
  「誰にも言えない悩みを抱えたとき、どうすればいいか」
  → note・SEO記事。「吐き出したい 方法」「愚痴 聞いてほしい」等のキーワード
  → Mintが1000本の記事でローンチ前にSEO1位を取った戦略の小規模版

③ SNSのオーガニック（コスト¥0）
  → 会話サンプルスクリーンショットをX/Instagramに投稿
  → 「こんな返答が来た」が口コミになる（Replikaが初期にやった戦略）

有料広告（ASA）は LTV/CPA > 3 確認後のみ
```

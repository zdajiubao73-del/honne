# App Store 審査提出準備

## 概要
App Store 審査に通過するための法的ドキュメント（プライバシーポリシー・利用規約）を作成し、App Store Connect のメタデータをすべて入力して提出できる状態にする。

## 背景
REQUIREMENTS.md §11 に審査対応チェックリストが定義されている。メンタルヘルスカテゴリは審査が厳しく、「医療機器ではない」明記・危機対応の実装・プライバシーポリシーの公開が審査官から必ず確認される。

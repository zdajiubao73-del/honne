# セーフガード統合（危機ワード検知 → 専門機関誘導）

## 概要
`SafetyFilter.swift` を `ChatViewModel` に統合し、ユーザーが危機的なワードを入力した際に即座に専門機関への誘導メッセージを表示する。App Store 審査にも必須の要件。

## 背景
REQUIREMENTS.md §3-1（F10）に「危機ワード検知」「専門機関への誘導」が Must Have として定義されている。また §9-4 で「危機時は必ず専門機関へ誘導し、会話を継続しない」とある。現在 `SafetyFilter.swift` は単体で存在するが、ChatViewModel への統合が未確認。

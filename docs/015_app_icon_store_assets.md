# アプリアイコン・ストアアセット作成

## 概要
App Store 提出に必要なアイコン・スクリーンショット・プレビュー動画を作成する。FIGMA_SPEC.md のアイコン仕様に基づきアイコンを設計し、Xcode に組み込む。

## 背景
REQUIREMENTS.md §11 に「iPhone 6.7インチのスクリーンショット5枚必須」「プレビュー動画30秒」が定義されている。FIGMA_SPEC.md §アプリアイコン仕様に「暗いグラデーション + 余白多めの白い半円」のデザインが定義されている。

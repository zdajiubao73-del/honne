# AI会話フェーズ管理ロジック（3ステップ実装）

## 概要
REQUIREMENTS.md §6 の「共感 → 深掘り → 整理」3ステップ会話モデルを ChatViewModel に実装する。フェーズに応じてシステムプロンプトの指示を動的に切り替え、AIが適切なタイミングで返答スタイルを変えるようにする。

## 背景
現在 `ChatViewModel.swift` は固定のシステムプロンプトを使っているが、要件では会話の往復数に応じてフェーズを進行させる必要がある。「共感フェーズ中に『どうしたいですか？』を言わない」などのフェーズ別禁止事項が最重要品質要件。

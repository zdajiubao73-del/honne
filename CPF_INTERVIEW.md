## KJ法によるインサイト集約

### グループA：「話せない場所の問題」（全10人共通）

```
言動: 「友達には弱みを見せたくない」「愚痴っぽくなるのが嫌」「格好悪い」
     「判断されたくない」「ママ友には競争感がある」

→ インサイト:「話す内容」ではなく「自己イメージの保護」が最大の障壁
→ honneが解決すること: AIなら「弱い自分・情けない自分」を見せても自己イメージが傷つかない
→ コピーへの示唆: 「誰にも知られない」より「自分らしくいられる場所」のほうが刺さるかも
```

### グループB：「一方通行では足りない」（10人中9人）

```
言動: 「日記に書いても返ってこない」「Twitterは一方通行」
     「考えがぐるぐるする」「整理できない」

→ インサイト: 「書く」ことへのニーズはあるが「返ってくる」ことで初めて完結する
→ honneが解決すること: 双方向の対話で「思考のループ」から抜け出せる
→ AIの返答品質がプロダクト価値の本体であることを再確認
```

### グループC：「時間・タイミングの制約」（10人中6人）

```
言動: 「友達と会うのは調整が大変」「深夜2時でも聞いてほしい」
     「夜勤明けに使いたい」「5分でいい」

→ インサイト: 「話したい瞬間」と「話せる機会」のズレが最大の問題
→ honneが解決すること: 24時間・非同期・5分から使える
→ コピーへの示唆: 「夜、誰かに話を聞いてほしいとき」は正確。深夜〜早朝のプッシュ通知設計も有効
```

### グループD：「AIへの受容性」（10人中8人が抵抗なし）

```
抵抗なし: ④翔太「全くない」⑧ゆか「むしろAIがいい」⑤あおい「抵抗はない」
         ⑥りな「ない」⑩はな「もう誰でもいい」

              ⑦大輔「情報漏洩リスク」⑨誠「情けない気がしてた」

→ インサイト: AIへの心理的抵抗より「本当に寄り添えるか」という機能的懸念が主
→ 解決策: 最初の会話体験の質が全て。App Storeのプレビュー動画・サンプル会話が超重要
→ 「AIと話す = 弱い・情けない」という先入観 → 特に40代男性向けにケアが必要
```

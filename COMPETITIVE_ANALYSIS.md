## 競合詳細分析

### 1. Awarefy（アウェアファイ）★ 最重要競合

| 項目 | 内容 |
|------|------|
| 運営 | 株式会社Awarefy（日本） |
| コンセプト | 認知行動療法（CBT）ベースのAIメンタルパートナー |
| DL数 | 累計70万+ |
| App Store評価 | ★4.3（5,000件超レビュー） |
| 資金調達 | 合計約6.2億円（プレシリーズA 2.2億円 + シリーズA 4億円 / 2024年12月） |
| MRR成長 | 1年2ヶ月でMRR約5倍以上 |
| 研究機関 | 早稲田大学と共同研究 |

**料金プラン**
```
無料プラン:      ¥0      AIチャットほぼ利用不可・機能大幅制限
ベーシックプラン: ¥1,600/月（年額 ¥9,600 = ¥800/月）
AIプラン:        ¥4,480/月（年額 ¥19,000 = ¥1,583/月）
```

**主な機能**
- AIメンタルサポーター「ファイさん」とのチャット（AIプランのみフル機能）
- 感情・体調・睡眠の記録
- CBT・マインドフルネスの学習コース
- こころの状態測定（PHQ-9等の心理尺度）
- AIメモリー・AIコーチング（AIプランのみ）

**ユーザーの声**
- 好評：「AIとのチャットで気軽に話を聞いてもらえる」「習慣化できる」
- 不満：「無料プランではAI機能がほぼ使えない」「CBTのワークが少し堅い」「AIの返答が表面的」

**honneとの差分**
```
Awarefy: CBT・自己理解のための「ツール」。ワーク・課題・学習を提供
honne:   ただ「吐き出して整理する」体験。課題設定なし・アドバイスなし
→ Awarefyは「改善したい人」向け。honneは「ただ話したい人」向け
```

## リスク分析：競合の動き

### 短期リスク（〜1年）
```
⚠️ Awarefy が「吐き出し特化モード」を追加してくる
   → 資金6.2億円・DL70万のAwarefyがピボットした場合、最大の脅威
   → 対策：早期にブランドを「本音 = honne」として確立する

⚠️ muute が双方向チャット機能を追加してくる
   → ★4.7・19,000件レビューを持つmuuteが会話機能を持てば強い競合に
   → 対策：会話の質（AIプロンプト設計）で先行する

⚠️ ChatGPT / Claude の直接利用
   → 「ChatGPTに愚痴を話す」ユーザーがすでにいる（フリーランス・翔太型）
   → 対策：「メンタル特化体験」と「感情の記録・可視化」で差別化
```

### 中期リスク（1〜3年）
```
⚠️ Wysa・Woebot の日本語本格参入
   → 臨床エビデンスと国際的なブランド力を持つ
   → 対策：日本固有の文化（本音と建前）に根ざしたプロンプト設計を深化

⚠️ LINEがメンタルヘルスチャット機能を追加
   → MAU9,500万人のLINEが参入したら普及速度は段違い
   → 対策：ニッチ特化・コアユーザーのロイヤリティを先に確保
```

## 戦略的示唆

### 1. App Store のカテゴリ選択
```
「ヘルスケア＆フィットネス」→ 競合が多い
「ライフスタイル」→ 競合が少なく発見されやすい可能性

推奨：まず「ヘルスケア＆フィットネス」でAwarefy・muuteと並べ、
     キーワード「本音」「悩み」「匿名」で独自性を出す
```

### 2. 差別化コピーの確定

競合と並んだときのhonneの一言：
```
Awarefy → 「心を整えるアプリ」
muute   → 「書いて、気づくアプリ」
honne   → 「判断されない場所で、話すアプリ」
```

### 3. Awarefy ユーザーの「不満」を獲得する

Awarefyのレビューで繰り返し出る不満：
- 「無料だとAIがほぼ使えない」
- 「CBTのワークが続かない」
- 「AIの返答が表面的・マニュアル的」

→ honneのApp Store説明文に「ワークなし。課題なし。ただ話すだけ」を入れる
→ Awarefyで挫折したユーザーがキーワード検索でhonneに辿り着く導線を設計

### 4. muute ユーザーの「次のステップ」になる

muuteのレビューで繰り返し出る不満：
- 「一方通行で返事が来ない」
- 「会話したい」

→ muuteのレビューに「もっと話したい方はhonne」という発信（SNS）
→ 「日記よりも話したい」というキーワードでSEO/ASO

### 5. 価格設定の優位性

```
Awarefy AIプラン: ¥4,480/月（年額換算¥1,583/月）
honne Pro:        ¥980/月（年額換算¥650/月）

→ Awarefyの AIプランの約1/4の価格で「話す体験」に特化
→ 「Awarefyより安くて、話すことに特化している」が最強の導線
```

## 結論：honneが勝てる理由

```
1. ポジションの空白を取っている
   「CBTでもなく・友達AIでもなく・人間でもない」日本語の吐き出し場所

2. 価格優位性がある
   最強競合Awarefyのフル機能プランの約1/4の価格

3. 匿名スタートが唯一無二
   全競合がアカウント作成を要求している（muute・emol・Awarefy）
   honneだけが「名前もメアドも不要」で即開始できる

4. タイミングが良い
   Awarefy が6.2億円を調達し市場の認知を作っている最中
   市場をAwarefyに開拓させて、「話したいだけ」層を獲りにいく
```
